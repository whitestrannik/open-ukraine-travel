## Navigation Design Pattern
### Pill-Style Navigation (Implemented)
- **Container**: Rounded full background (`bg-gray-50/80 rounded-full px-2 py-1`)
- **Active State**: Full blue background with shadow (`bg-[#1F5FA0] shadow-lg shadow-[#1F5FA0]/25`)
- **Hover Effects**: Scale transform (`hover:scale-105`) + gradient background
- **Typography**: Medium font weight, proper spacing
- **Responsive**: Collapses to hamburger on mobile

## Card Design Pattern
- **Shadow**: `shadow-lg hover:shadow-xl`

- **Video Background**: Full viewport height with overlay

## Responsive Breakpoints
- **Mobile**: Default (mobile-first)

- **Transform Effects**: `hover:-translate-y-2` for card lifts
